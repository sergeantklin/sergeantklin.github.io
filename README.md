# sergeantklin.github.io

Personal portfolio website of Sergei Kalinkovskii — Lead Frontend Developer.

## About

A bilingual (RU/EN) single-page application built with React, TypeScript, and Tailwind CSS. Deployed on GitHub Pages.

**Sections:**
- **About** — short bio, photo, and links to GitHub, Telegram, and Email
- **Experience** — work history with company logos, tech stack tags, responsibilities and achievements
- **Code** — TypeScript code samples with animated syntax highlighting
- **Hobbies** — interests presented as icon cards

**Features:**
- RU/EN language toggle — switches all content on the fly
- Light theme with subtle sky/purple gradient background
- Smooth page transition animations (Framer Motion)
- Responsive layout, bottom navigation bar

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router (HashRouter for GitHub Pages compatibility)
- Framer Motion
- Lucide React

## Development

```bash
npm install
npm run dev
```

## Deploy

The site is deployed automatically to GitHub Pages from the `master` branch.

```bash
npm run build
```
